# 市场营销插件

专为OpenClaw设计的营销插件——支持内容创作、活动策划、品牌声音管理、竞争分析和效果报告。

## 安装

```bash
openclaw plugins add knowledge-work-plugins/marketing
```

## 命令

| 命令 | 描述 |
|------|------|
| `/draft-content` | 起草博客文章、社交媒体帖子、邮件简报、落地页、新闻稿和案例研究 |
| `/campaign-plan` | 生成含目标、渠道策略、内容日历和成功指标的完整活动简报 |
| `/brand-review` | 对照品牌声音、风格指南和信息支柱审查内容 |
| `/competitive-brief` | 调研竞争对手，生成定位和信息对比分析 |
| `/performance-report` | 生成含关键指标、趋势和优化建议的营销效果报告 |
| `/seo-audit` | 进行全面SEO审查——关键词调研、页面分析、内容差距、技术检查和竞争对手对比 |
| `/email-sequence` | 设计并起草多封邮件序列，适用于培育、引导、滴灌活动等 |

## 技能

| 技能 | 描述 |
|------|------|
| `content-creation` | 各渠道内容类型模板、写作最佳实践、SEO基础、标题公式和行动号召指南 |
| `campaign-planning` | 活动框架、渠道选择、内容日历创建、预算分配和成功指标 |
| `brand-voice` | 品牌声音文档框架、声音属性、语气调整、风格规范执行和术语管理 |
| `competitive-analysis` | 竞争调研方法论、信息对比、内容差距分析、定位和竞争销售手册创建 |
| `performance-analytics` | 各渠道关键指标、报告模板、趋势分析、归因建模和优化框架 |

## 典型工作流程示例

### 起草博客文章

```
> /draft-content
类型：博客文章
主题：AI如何改变B2B营销
受众：中型SaaS公司的市场总监
核心信息：AI节省重复性任务时间、改善个性化、需要人工监督
语气：权威但易于接近
长度：1200字
```

AI助手将生成结构完整的博客文章草稿，包含吸引人的标题、有钩子的导语、有组织的章节、SEO优化的小标题和清晰的行动号召。

### 策划营销活动

```
> /campaign-plan
目标：为新产品发布推动500个注册
受众：企业级公司的技术决策者
时间：6周
预算范围：20,000-30,000美元
```

AI助手将生成活动简报，涵盖目标、受众细分、核心信息、渠道策略、按周划分的内容日历和需追踪的KPI。

### 对照品牌规范审查内容

```
> /brand-review
[粘贴您的内容草稿]
```

如果您的品牌风格指南已在本地设置中配置，AI助手将对照声音、语气、术语和信息支柱检查您的内容。如未配置，AI助手将询问您的规范或提供清晰度、一致性和专业性的通用审查。

## 配置

在本地设置文件中配置您的品牌声音、风格指南和目标用户画像，以获得个性化输出。这样，`/draft-content` 和 `/brand-review` 等命令每次都会自动应用您的品牌标准，无需重复说明。

## MCP集成

> 如遇不熟悉的占位符或需要查看已连接工具，请参见 [CONNECTORS.md](CONNECTORS.md)。如需配置MCP服务，请参见 [MCP_SETUP.md](MCP_SETUP.md)。

本插件支持以下MCP服务：

- **Slack** ——与团队分享草稿、报告和简报
- **Canva** ——创建和编辑设计资产
- **Figma** ——访问设计文件和品牌资产
- **HubSpot** ——拉取活动数据、管理联系人和追踪营销自动化
- **Amplitude** ——拉取产品分析和用户行为数据用于效果报告
- **Notion** ——访问简报、风格指南和活动文档
- **Ahrefs** ——SEO关键词调研、反向链接分析和站点审查
- **Similarweb** ——竞争对手流量分析和市场对标
- **Klaviyo** ——起草和管理邮件营销序列
- **Supermetrics** ——整合多平台营销数据用于分析和报告
