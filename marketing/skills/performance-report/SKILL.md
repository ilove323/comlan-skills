# 效果报告

> 如遇不熟悉的占位符或需要查看已连接工具，请参见 [CONNECTORS.md](../../CONNECTORS.md)。

生成含关键指标、趋势分析、洞察和优化建议的营销效果报告。

## 触发方式

用户运行 `/performance-report` 或要求营销报告、效果分析、活动结果或指标摘要。

## 输入

1. **报告类型** ——确认用户需要哪种报告：
   - **活动报告** ——特定活动的效果
   - **渠道报告** ——特定渠道的效果（邮件、社交、付费、SEO等）
   - **内容效果** ——内容资产的表现
   - **整体营销报告** ——跨渠道汇总（周报、月报、季报）
   - **自定义** ——用户自定义范围

2. **时间段** ——报告窗口（上周、上月、上季度、自定义日期范围）

3. **数据来源**：
   - 若已连接 ~~marketing analytics，先了解哪些账户和平台可用，然后自动拉取效果数据
   - 若已连接 ~~product analytics：自动拉取效果数据
   - 若均未连接：请求用户提供数据，提示如下："请粘贴或分享您的效果数据。可以是电子表格、CSV数据、用文字描述的仪表板截图，或仅提供关键数字。"

4. **对比期间**（可选）——用于趋势背景的上一期或同比数据

5. **利益相关方受众**（可选）——谁将阅读此报告（高管摘要风格 vs. 详细分析视角）

## 报告结构

### 1. 执行摘要
- 2-3句话概述本期表现
- 附趋势方向的主要指标（与上期相比上升/下降/持平）
- 一个关键亮点和一个关注点

### 2. 关键指标看板

以汇总表格呈现核心指标：

| 指标 | 本期 | 上期 | 变化 | 目标 | 状态 |
|------|------|------|------|------|------|

状态标识：
- 达标（达到或超过目标）
- 有风险（低于目标但在可接受范围内）
- 未达标（显著低于目标）

#### 按报告类型划分的指标

**活动报告：**
- 曝光量和覆盖率
- 点击率（CTR）
- 转化率
- 每次获客成本（CPA）
- 广告支出回报率（ROAS）或ROI
- 总转化/注册/线索数

**渠道报告（邮件）：**
- 发送量、送达量、退信量
- 打开率
- 点击率
- 退订率
- 转化率

**渠道报告（社交）：**
- 曝光量和覆盖率
- 互动率（点赞、评论、分享）
- 粉丝增长
- 点击率
- 热门帖子

**渠道报告（付费）：**
- 支出
- 曝光量和点击量
- 点击率
- CPC和CPM
- 转化量和CPA
- ROAS

**渠道报告（SEO/自然搜索）：**
- 自然搜索会话数
- 关键词排名变化
- 已索引页面数
- 获取的反向链接
- 表现最佳的页面

**内容效果：**
- 页面浏览量和独立访客
- 页面停留时长
- 跳出率
- 社交分享数
- 归因至内容的转化数
- 表现最佳和最差的内容

**整体营销报告：**
- 产生的总线索数
- 市场合格线索（MQL）
- 商机贡献
- 客户获取成本（CAC）
- 各渠道汇总

### 3. 趋势分析
- 期间内的效果趋势（周环比或月环比）
- 显著的拐点及其原因
- 观察到的季节性或周期性规律
- 与基准或目标的对比

### 4. 亮点
- 3-5个附具体数据的亮点
- 为什么这些表现好（假设）
- 如何复制或扩大

### 5. 待改进点
- 3-5个附具体数据的表现不佳项
- 表现不佳的假设原因
- 建议改进措施

### 6. 洞察与观察
- 仅看指标不够直观的数据规律
- 用户行为洞察
- 引发共鸣的内容或创意主题
- 可能影响效果的外部因素（季节性、新闻事件、竞争动作）

### 7. 优化建议
对每项建议：
- 做什么
- 为什么（关联数据中的具体洞察）
- 预期影响（高/中/低）
- 实施难度（高/中/低）
- 优先级（立即/下个冲刺/下个季度）

以2×2矩阵格式呈现优先级建议：

| | 低难度 | 高难度 |
|---|---|---|
| **高影响** | 优先执行 | 纳入下个冲刺计划 |
| **低影响** | 有时间再做 | 降低优先级 |

### 8. 下期重点
- 下个周期的3个核心优先事项
- 需要运行的测试或实验
- 关键指标目标

## 指标定义与基准

### 邮件营销

| 指标 | 定义 | 基准范围 | 说明 |
|------|------|---------|------|
| 送达率 | 送达邮件数/发送邮件数 | 95-99% | 列表健康度和发件人信誉 |
| 打开率 | 独立打开数/送达邮件数 | 15-30% | 主题行和发件人有效性 |
| 点击率（CTR） | 独立点击数/送达邮件数 | 2-5% | 内容相关性和行动号召有效性 |
| 点开率（CTOR） | 独立点击数/独立打开数 | 10-20% | 邮件内容质量（对已打开的人） |
| 退订率 | 退订数/送达邮件数 | <0.5% | 内容与受众匹配度及频率 |
| 退信率 | 退信数/发送邮件数 | <2% | 列表质量和数据卫生 |
| 转化率 | 转化数/送达邮件数 | 1-5% | 端到端邮件效果 |

### 社交媒体

| 指标 | 定义 | 说明 |
|------|------|------|
| 曝光量 | 内容展示次数 | 内容分发和触达范围 |
| 覆盖率 | 看到内容的独立用户数 | 受众广度 |
| 互动率 | (点赞+评论+分享)/覆盖率 | 内容共鸣度 |
| 点击率 | 链接点击量/曝光量 | 导流效果 |
| 粉丝增长率 | 每期净增粉丝/总粉丝 | 受众积累 |
| 分享/转发率 | 分享数/覆盖率 | 内容病毒性和传播力 |
| 视频播放率 | 播放数/曝光量 | 视频钩子效果 |
| 视频完播率 | 完整播放数/总播放数 | 视频质量和时长适配 |

### 付费广告

| 指标 | 定义 | 说明 |
|------|------|------|
| 曝光量 | 广告展示次数 | 预算利用率和定向广度 |
| 点击率（CTR） | 点击数/曝光量 | 广告创意和定向相关性 |
| 每次点击成本（CPC） | 总支出/点击数 | 流量获取成本效率 |
| 每千次展示成本（CPM） | 每千次展示的成本 | 认知成本效率 |
| 转化率 | 转化数/点击数 | 落地页和优惠有效性 |
| 每次获客成本（CPA） | 总支出/转化数 | 全漏斗成本效率 |
| 广告支出回报率（ROAS） | 收入/广告支出 | 收入产生效率 |

### SEO/自然搜索

| 指标 | 定义 | 说明 |
|------|------|------|
| 自然搜索会话数 | 来自自然搜索的访问 | SEO效果和内容触达 |
| 关键词排名 | 目标关键词的排名位置 | 搜索可见度 |
| 自然搜索点击率 | 搜索结果中的点击数/曝光量 | 标题和Meta描述效果 |
| 反向链接数 | 链接到您的外部网站数量 | 内容权威性和外链SEO |
| 自然搜索转化率 | 自然搜索转化数/自然搜索会话数 | 内容质量和意图匹配 |

### 整体营销/商机

| 指标 | 定义 | 说明 |
|------|------|------|
| 市场合格线索（MQL） | 达到营销资格标准的线索 | 顶部漏斗效果 |
| 销售合格线索（SQL） | 销售接受的MQL | 线索质量 |
| MQL到SQL转化率 | SQL/MQL | 市场销售协同和线索质量 |
| 产生商机 | 创造的销售机会金额 | 营销对收入的影响 |
| 商机流速 | 商机在漏斗中的推进速度 | 活动紧迫性和质量 |
| 客户获取成本（CAC） | 市场+销售总成本/新客户数 | 客户获取效率 |
| 市场来源收入 | 源自营销的收入 | 营销的直接贡献 |

## 按频次划分的报告模板

### 周报
团队站会快速浏览格式：
- **3个核心指标**及周环比变化
- **本周亮点**（1-2条带数据）
- **需要关注**（1-2条带数据）
- **本周优先事项**（3-5项行动）

### 月报
标准利益相关方报告：
1. 执行摘要（3-5句话）
2. 关键指标看板（含月环比和目标对比的表格）
3. 各渠道效果摘要
4. 活动亮点和成果
5. 亮点和待改进点（含假设）
6. 建议和下月优先事项
7. 实际支出 vs. 计划

### 季度业务回顾（QBR）
高管战略回顾：
1. 本季度 vs. 目标表现
2. 年初至今轨迹
3. 各渠道ROI分析
4. 活动效果摘要
5. 竞争和市场观察
6. 下季度战略建议
7. 预算申请和分配计划
8. 关键实验和经验教训

- 深入分析某个具体指标或渠道？
- 设置可下期复用的报告模板？"
