# /runbook

> 如果您看到不熟悉的占位符或需要检查已连接的工具，请参阅 [CONNECTORS.md](../../CONNECTORS.md)。

为定期任务或操作程序创建分步骤的操作运行手册。

## 用法

```
/runbook $ARGUMENTS
```

## 使用技巧

1. **极度具体** — "运行脚本"不是一个步骤。"从运营服务器执行 `python sync.py --prod --dry-run`"才是。
2. **包含失败模式** — 每个步骤可能出现什么问题以及如何处理。
3. **测试运行手册** — 让一个不熟悉流程的人按照它操作。修复他们卡住的地方。
