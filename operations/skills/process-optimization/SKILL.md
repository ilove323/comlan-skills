# 流程优化

分析现有流程并提出改进建议。

## 分析框架

### 1. 绘制现状图
- 记录每个步骤、决策点和交接环节
- 确定谁在做什么以及每个步骤需要多长时间
- 标注手动步骤、审批环节和等待时间

### 2. 识别浪费
- **等待**：在队列中或等待审批的时间
- **返工**：失败并需要重做的步骤
- **交接**：每次交接都是潜在的失败或延迟点
- **过度处理**：不增加价值的步骤
- **手动工作**：可以自动化的任务

### 3. 设计未来状态
- 消除不必要的步骤
- 尽可能自动化
- 减少交接次数
- 并行处理独立步骤
- 添加检查点（而非关卡）

### 4. 衡量影响
- 每个周期节省的时间
- 错误率降低
- 成本节约
- 员工满意度提升

## 输出

生成流程前后对比，附带具体改进建议、预估影响和实施计划。
