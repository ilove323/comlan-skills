# 统计分析技能

描述性统计、趋势分析、异常检测、假设检验，以及何时对统计结论保持谨慎的指导。

## 描述性统计方法论

### 集中趋势

根据数据选择合适的集中趋势度量：

| 情况 | 使用 | 原因 |
|---|---|---|
| 对称分布，无异常值 | 均值 | 最有效的估计量 |
| 偏斜分布 | 中位数 | 对异常值具有鲁棒性 |
| 分类或有序数据 | 众数 | 非数值数据的唯一选项 |
| 高度偏斜且有异常值（如每用户营收） | 中位数 + 均值 | 同时报告；两者差距反映偏斜程度 |

**业务指标务必同时报告均值和中位数。** 若两者差异显著，数据存在偏斜，单独使用均值会产生误导。

### 离散程度与变异性

- **标准差**：值偏离均值的典型程度。适用于正态分布数据。
- **四分位距（IQR）**：p25 到 p75 的距离。对异常值具有鲁棒性。适用于偏斜数据。
- **变异系数（CV）**：标准差/均值。用于比较不同量纲指标之间的变异程度。
- **极差**：最大值减最小值。对异常值敏感，但能快速感知数据范围。

### 业务背景下的分位数

报告关键分位数，比单纯使用均值讲述更丰富的故事：

```
p1：最低 1%（下限 / 最小典型值）
p5：正常范围的低端
p25：第一四分位数
p50：中位数（典型用户）
p75：第三四分位数
p90：前 10% / 高活跃用户
p95：正常范围的高端
p99：前 1% / 极端用户
```

**示例叙述**："会话时长的中位数为 4.2 分钟，但前 10% 的用户每次会话超过 22 分钟，将均值拉升至 7.8 分钟。"

### 描述分布

分析每个数值分布时，描述以下特征：

- **形状**：正态、右偏、左偏、双峰、均匀、重尾
- **集中**：均值和中位数（及两者差距）
- **离散**：标准差或四分位距
- **异常值**：数量和极端程度
- **边界**：是否有自然下限（零）或上限（100%）？

## 趋势分析与预测

### 识别趋势

**移动平均**平滑噪声：
```python
# 7日移动平均（适合具有周季节性的每日数据）
df['ma_7d'] = df['metric'].rolling(window=7, min_periods=1).mean()

# 28日移动平均（同时平滑周和月度规律）
df['ma_28d'] = df['metric'].rolling(window=28, min_periods=1).mean()
```

**同比/环比对比**：
- 周环比（WoW）：与上周同日对比
- 月环比（MoM）：与上月同月对比
- 年同比（YoY）：季节性业务的黄金标准
- 去年同日：对比具体日历日

**增长率**：
```
简单增长率：（当期 - 上期）/ 上期
CAGR：（终值 / 初值）^ （1 / 年数）- 1
对数增长：ln（当期 / 上期）—— 对波动较大的序列更合适
```

### 季节性检测

检查周期性规律：
1. 绘制原始时间序列——先进行目视检查
2. 计算各星期几的均值：是否存在明显的周度规律？
3. 计算各月份均值：是否存在年度周期？
4. 对比周期时，始终使用同比或同期对比，避免将趋势与季节性混淆

### 预测（简单方法）

对于业务分析人员（非数据科学家），使用直观的方法：

- **朴素预测**：明天 = 今天。用作基准。
- **季节性朴素**：明天 = 上周/去年同日。
- **线性趋势**：对历史数据拟合直线。仅适用于明显的线性趋势。
- **移动平均预测**：用近期均值作为预测值。

**始终传达不确定性。** 提供区间而非点估计：
- "基于3个月趋势，我们预计下月新增 1万~1.2万 注册用户"
- 不要说"我们下月将精确获得 11,234 个注册用户"

**何时升级到数据科学家**：非线性趋势、多重季节性、外部因素（营销费用、节假日），或当预测准确度对资源分配至关重要时。

## 异常值与异常检测

### 统计方法

**Z 值法**（适用于正态分布数据）：
```python
z_scores = (df['value'] - df['value'].mean()) / df['value'].std()
outliers = df[abs(z_scores) > 3]  # 超过3个标准差
```

**IQR 法**（对非正态分布具有鲁棒性）：
```python
Q1 = df['value'].quantile(0.25)
Q3 = df['value'].quantile(0.75)
IQR = Q3 - Q1
lower_bound = Q1 - 1.5 * IQR
upper_bound = Q3 + 1.5 * IQR
outliers = df[(df['value'] < lower_bound) | (df['value'] > upper_bound)]
```

**百分位法**（最简单）：
```python
outliers = df[(df['value'] < df['value'].quantile(0.01)) |
              (df['value'] > df['value'].quantile(0.99))]
```

### 处理异常值

**不要**自动删除异常值，而是：

1. **调查**：这是数据错误、真实的极端值，还是不同的人群？
2. **数据错误**：修正或删除（例如，负年龄、1970年的时间戳）
3. **真实极端值**：保留，但考虑使用鲁棒统计量（中位数代替均值）
4. **不同人群**：将其单独分出做独立分析（例如，企业客户 vs. 中小企业客户）

**说明你的做法**："我们排除了 47 条记录（0.3%），这些交易金额超过 5万 元，代表独立分析的大宗企业订单。"

### 时间序列异常检测

检测时间序列中的异常值：

1. 计算期望值（移动平均或去年同期）
2. 计算与期望值的偏差
3. 标记超过阈值（通常为残差标准差的 2-3 倍）的偏差
4. 区分点异常（单个异常值）和拐点（持续性变化）

## 假设检验基础

### 使用场景

当需要判断观察到的差异是否可能真实存在或只是随机波动时，使用假设检验。常见场景：

- A/B 测试结果：变体 B 真的比 A 更好吗？
- 前后对比：产品变更是否真的改变了指标？
- 细分对比：企业客户的留存率真的更高吗？

### 框架

1. **零假设（H0）**：不存在差异（默认假设）
2. **备择假设（H1）**：存在差异
3. **选择显著性水平（α）**：通常为 0.05（5% 的假阳性概率）
4. **计算检验统计量和 p 值**
5. **解读**：若 p < α，拒绝 H0（有真实差异的证据）

### 常用检验

| 场景 | 检验方法 | 适用时机 |
|---|---|---|
| 比较两组均值 | 独立样本 t 检验 | 正态数据，两组 |
| 比较两组比例 | 比例 z 检验 | 转化率、二元结果 |
| 比较配对测量 | 配对 t 检验 | 同一实体的前后对比 |
| 比较三组以上均值 | 方差分析（ANOVA） | 多个细分或变体 |
| 非正态数据，两组 | Mann-Whitney U 检验 | 偏斜指标、有序数据 |
| 类别变量之间的关联 | 卡方检验 | 两个分类变量 |

### 统计显著性 vs. 实际显著性

**统计显著性**意味着差异不太可能由随机因素造成。

**实际显著性**意味着差异足够大，对业务决策有意义。

在大样本情况下，差异可以在统计上显著但在实践中毫无意义。始终报告：
- **效应量**：差异有多大？（例如，"变体B将转化率提升了0.3个百分点"）
- **置信区间**：真实效果的合理范围是什么？
- **业务影响**：这转化为多少营收、用户或其他业务指标？

### 样本量考虑

- 小样本产生不可靠的结果，即使 p 值显著
- 比例数据的经验法则：每组至少需要30个事件才能保证基本可靠性
- 要检测微小效应（例如1%的转化率变化），每组可能需要数千个观测
- 如果样本量较小，明确说明："每组只有200个观测，我们检测小于X%效应的统计功效有限"

## 对统计结论保持谨慎的时机

### 相关性不等于因果关系

发现相关性时，明确考虑：
- **反向因果**：可能是B导致了A，而不是A导致B
- **混淆变量**：可能是C同时导致了A和B
- **巧合**：有足够多的变量，虚假相关是不可避免的

**可以说的**："使用功能X的用户留存率高30%"
**在没有更多证据的情况下不能说的**："功能X导致了30%更高的留存率"

### 多重比较问题

当测试许多假设时，某些会因偶然因素而"显著"：
- 在 p=0.05 水平测试20个指标，大约1个会是假阳性
- 如果在找到一个不同的细分之前查看了很多细分，请注明
- 使用 Bonferroni 校正（α除以测试数量）或报告测试了多少个假设

### 辛普森悖论

汇总数据中的趋势在数据分细分后可能反转：
- 始终检查结论在关键细分中是否成立
- 示例：整体转化率上升，但每个细分的转化率都在下降——因为结构向转化率更高的细分偏移了

### 幸存者偏差

你只能分析"幸存"进入数据集的实体：
- 分析"活跃用户"忽略了已流失的用户
- 分析"使用我们产品的公司"忽略了那些评估后离开的公司
- 研究"成功"结果的特征，不包含"失败"结果
- 始终问："谁不在这个数据集里？把他们纳入会改变结论吗？"

### 生态谬误

群体趋势不一定适用于个体：
- "X较高的国家Y也较高"并不意味着"X较高的个人Y也较高"
- 将群体层面的发现应用于个体案例时要谨慎

### 过度依赖精确数字

警惕虚假精确：
- "下季度流失率将为4.73%"暗示的确定性超过了实际情况
- 首选区间："基于历史规律，我们预计流失率在4%-6%之间"
- 适当取整："约5%"通常比"4.73%"更诚实
