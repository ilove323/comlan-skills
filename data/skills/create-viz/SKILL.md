# /create-viz - 创建可视化图表

> 如遇到不熟悉的占位符或需要查看哪些工具已接入，请参阅 [CONNECTORS.md](../../CONNECTORS.md)。

使用 Python 创建出版级数据可视化。根据数据生成图表，遵循清晰度、准确性和设计最佳实践。

## 使用方式

```
/create-viz <数据来源> [图表类型] [其他说明]
```

## 工作流

### 1. 理解需求

确定：

- **数据来源**：查询结果、粘贴的数据、CSV/Excel 文件，或待查询的数据
- **图表类型**：明确指定，或需要推荐
- **用途**：探索分析、演示汇报、报告、仪表板组件
- **受众**：技术团队、高管、外部干系人

### 2. 获取数据

**如果已接入数据仓库且需要查询数据：**
1. 编写并执行查询
2. 将结果加载为 pandas DataFrame

**如果数据是粘贴或上传的：**
1. 将数据解析为 pandas DataFrame
2. 根据需要清洗和准备（类型转换、空值处理）

**如果数据来自对话中的前一步分析：**
1. 引用已有数据

### 3. 选择图表类型

如果用户未指定图表类型，根据数据和问题推荐一种：

| 数据关系 | 推荐图表 |
|---|---|
| 随时间的趋势 | 折线图 |
| 类别间的对比 | 柱状图（类别较多时用水平方向） |
| 整体中的部分构成 | 堆叠柱状图或面积图（避免饼图，除非类别少于6个） |
| 值的分布 | 直方图或箱形图 |
| 两变量的相关性 | 散点图 |
| 随时间的双变量对比 | 双轴折线图或分组柱状图 |
| 地理数据 | 分级统计图 |
| 排名 | 水平柱状图 |
| 流向或过程 | 桑基图 |
| 关系矩阵 | 热力图 |

如果用户未指定，简要说明推荐理由。

### 4. 生成可视化图表

根据需求，使用以下 Python 库编写代码：

- **matplotlib + seaborn**：最适合静态出版级图表。默认首选。
- **plotly**：最适合交互式图表，或用户明确要求交互功能时。

**代码要求：**

```python
import matplotlib.pyplot as plt
import seaborn as sns
import pandas as pd

# 设置专业样式
plt.style.use('seaborn-v0_8-whitegrid')
sns.set_palette("husl")

# 创建适当尺寸的图形
fig, ax = plt.subplots(figsize=(10, 6))

# [特定图表代码]

# 始终包含：
ax.set_title('清晰、描述性的标题', fontsize=14, fontweight='bold')
ax.set_xlabel('X 轴标签', fontsize=11)
ax.set_ylabel('Y 轴标签', fontsize=11)

# 数字格式化规范
# - 百分比：'45.2%' 而非 '0.452'
# - 货币：'1.2M' 而非 '1200000'
# - 大数字：'2.3K' 或 '1.5M' 而非 '2300' 或 '1500000'

# 去除图表噪音
ax.spines['top'].set_visible(False)
ax.spines['right'].set_visible(False)

plt.tight_layout()
plt.savefig('chart_name.png', dpi=150, bbox_inches='tight')
plt.show()
```

### 5. 应用设计最佳实践

**颜色：**
- 使用一致的、色盲友好的调色板
- 有目的地使用颜色（而非装饰性使用）
- 用对比色突出关键数据点或趋势
- 将不重要的参考数据设为灰色

**文字：**
- 标题描述洞察而非仅描述指标（如 "营收同比增长 23%" 而非 "按月营收"）
- 坐标轴标签可读（尽量避免 90 度旋转）
- 在增加清晰度时在关键点添加数据标签

**布局：**
- 适当的空白和边距
- 图例位置不遮挡数据
- 按值排序类别（而非按字母序），除非有自然顺序

**准确性：**
- 柱状图 Y 轴从零开始
- 无误导性轴中断（如有则清晰标注）
- 对比多个子图时保持一致的刻度
- 适当的精度（不要显示 10 位小数）

### 6. 保存并呈现

1. 将图表保存为带描述性名称的 PNG 文件
2. 向用户展示图表
3. 提供所用代码，便于修改
4. 建议变体（不同图表类型、不同分组维度、缩放时间范围）
