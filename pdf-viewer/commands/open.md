# 打开PDF

在实时查看器中显示PDF文档。当用户想要**查看**文档而非仅提取文本时使用此命令。

## 操作步骤

1. 若用户提供了URL或文件路径，使用该路径调用`display_pdf`
2. 若未提供路径，先调用`list_pdfs`显示可用文档
3. 显示后，根据文档类型提供后续步骤建议：
   - **合同/报告** → "需要我高亮关键章节或添加审阅注释吗？"
   - **表单** → "此文档包含可填写字段——需要我协助填写吗？"
   - **学术论文** → "需要我逐步浏览并注释关键发现吗？"

## 支持的来源

- 本地文件（路径或拖放到工作目录）
- arXiv（`arxiv.org/abs/...` 自动转换为PDF链接）
- 任何直接的HTTPS PDF链接（使用PDF直链，而非落地页）

## 不适用情况

若用户只需摘要或文本提取，**请勿**打开查看器——改用AI助手的原生Read工具处理PDF路径。查看器用于交互式可视化工作流。
