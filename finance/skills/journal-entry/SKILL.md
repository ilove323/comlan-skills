# 日记账凭证编制

> 如遇不熟悉的占位符或需要查看已连接工具，请参见 [CONNECTORS.md](../../CONNECTORS.md)。

**重要提示**：本功能协助完成日记账凭证工作流程，不提供财务建议。所有凭证在过账前均应由具备资质的财务专业人员审核。

编制含借贷方向、支持明细和审核文档的日记账凭证。

## 用法

```
/je <类型> <期间>
```

### 参数

- `type` — 日记账凭证类型，可选：
  - `ap-accrual` — 已收货/服务但尚未开票的应付账款应计
  - `fixed-assets` — 固定资产和无形资产的折旧与摊销分录
  - `prepaid` — 预付费用摊销（保险、软件、租金等）
  - `payroll` — 薪资应计，含工资、福利、税款和奖金应计
  - `revenue` — 收入确认分录，含递延收入调整
- `period` — 会计期间（如 `2024-12`、`2024-Q4`、`2024`）

## 工作流程

### 1. 收集源数据

若已连接 ~~erp 或 ~~data warehouse：
- 拉取指定期间的试算平衡表
- 拉取相关科目的子账明细
- 拉取同类型上期凭证作为参考
- 识别受影响科目的当前总账余额

若无数据源连接：
> 请连接 ~~erp 或 ~~data warehouse 以自动拉取总账数据，也可粘贴试算平衡表数据或上传电子表格。

提示用户提供：
- 受影响科目的试算平衡表或总账余额
- 子账明细或支持性明细表
- 上期参考凭证（可选）

### 2. 计算凭证金额

根据凭证类型：

**应付账款应计：**
- 识别期末已收货/服务但尚未开票的项目
- 根据采购订单收货单、合同或估算计算应计金额
- 借：费用科目（或资本化项目记资产科目）
- 贷：应计负债

**固定资产：**
- 提取固定资产台账或折旧明细表
- 按资产类别和方法（直线法、加速折旧法、产量法）计算期间折旧
- 借：折旧费用（按部门/成本中心）
- 贷：累计折旧

**预付费用：**
- 提取预付费用摊销明细表
- 计算每项预付项目的期间摊销额
- 借：费用科目（按类型——保险、软件、租金等）
- 贷：预付费用科目

**薪资：**
- 计算已工作但尚未支付天数对应的应计工资
- 计算应计福利（医疗、退休金、带薪假）
- 计算雇主方工资税应计
- 计算奖金应计（如适用，按计划条款）
- 借：工资费用、福利费用、工资税费用
- 贷：应计工资、应计福利、应计工资税

**收入：**
- 审查合同和履约义务
- 根据交付/履约情况计算应确认收入
- 调整递延收入余额
- 借：递延收入（或应收账款）
- 贷：收入科目（按收入流/类别）

### 3. 生成日记账凭证

以标准格式列示凭证：

```
日记账凭证：[类型] — [期间]
编制人：[用户]
日期：[期末日期]

| 行 | 科目代码 | 科目名称 | 借方 | 贷方 | 部门 | 摘要 |
|----|---------|---------|------|------|------|------|
| 1  | XXXX    | [名称]  | X,XXX |      | [部门] | [明细] |
| 2  | XXXX    | [名称]  |       | X,XXX | [部门] | [明细] |
|    |         | **合计** | X,XXX | X,XXX |        |      |

支持明细：
- [计算依据和假设]
- [支持性明细表或文件的参考引用]

自动冲回：[是/否——如是，请注明冲回日期]
```

### 4. 复核清单

定稿前请核实：

- [ ] 借方合计等于贷方合计
- [ ] 会计期间正确
- [ ] 科目代码有效且对应正确的总账科目
- [ ] 金额经支持明细核实计算正确
- [ ] 摘要/描述清晰具体，足以支持审计
- [ ] 部门/成本中心编码正确
- [ ] 处理方式与上期一致
- [ ] 自动冲回标志设置适当（应计类凭证应自动冲回）
- [ ] 已引用或附上支持性文件
- [ ] 凭证金额在用户的审批权限内
- [ ] 无需要调查的异常或超出常规的金额

### 5. 输出

提供：
1. 格式化的日记账凭证
2. 支持性计算
3. 与同类型上期凭证的对比（如有）
4. 任何需要复核或跟进的标记项
5. 过账操作说明（手工录入或用户ERP的上传格式）
