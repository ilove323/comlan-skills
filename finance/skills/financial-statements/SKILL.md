# /financial-statements

> 如遇不熟悉的占位符或需要查看已连接工具，请参见 [CONNECTORS.md](../../CONNECTORS.md)。

**重要提示**：本功能协助完成财务报表工作流程，不提供财务建议。所有报表在用于报告或申报前，均应由具备资质的财务专业人员审核。

生成含期间对比和差异分析的财务报表。以下工作流程介绍利润表的生成过程；资产负债表和现金流量表的参考格式、GAAP列报要求（ASC 220/210/230）以及常见期末调整均作为参考资料附于文后。

## 用法

```
/financial-statements <期间类型> <期间>
```

### 参数

- `period-type` — 报告期间类型：
  - `monthly` — 单月利润表，含上月及去年同期对比
  - `quarterly` — 季度利润表，含上季度及去年同季度对比
  - `annual` — 全年利润表，含上年同期对比
  - `ytd` — 年初至今利润表，含上年同期YTD对比
- `period` — 报告期间（如 `2024-12`、`2024-Q4`、`2024`）

## 工作流程

### 1. 收集财务数据

若已连接 ~~erp 或 ~~data warehouse：
- 拉取指定期间的试算平衡表或利润表数据
- 拉取对比期间数据（上期、去年同期、预算/预测）
- 拉取账户层级和科目分组用于列报

若无数据源连接：
> 请连接 ~~erp 或 ~~data warehouse 以自动拉取财务数据，也可粘贴试算平衡表数据、上传电子表格或提供利润表数据进行分析。

提示用户提供：
- 当期收入和费用数据（按科目或类别）
- 对比期间数据（上期、去年同期和/或预算）
- 已知的调整或重分类

### 3. 差异分析

对每个行项目计算并标记重大差异。

#### 差异计算

对每个行项目计算：
- **金额差异：** 本期 - 上期（或本期 - 预算）
- **百分比差异：** （本期 - 上期）/ |上期| x 100
- **基点变动：** 对于利润率和比率，用基点表示变动（1 bp = 0.01%）

#### 重要性阈值

确定哪些差异构成需要调查的"重大"差异。常见方式：

- **固定金额阈值：** 差异超过设定金额（如5万元、10万元）
- **百分比阈值：** 差异超过设定百分比（如10%、15%）
- **组合方式：** 金额或百分比任一超过阈值即触发
- **分级方式：** 对不同行项目根据其规模和波动性设定不同阈值

*示例阈值（按组织情况调整）：*

| 行项目规模 | 金额阈值 | 百分比阈值 |
|----------|---------|-----------|
| > 1,000万 | 50万 | 5% |
| 100万 - 1,000万 | 10万 | 10% |
| < 100万 | 5万 | 15% |

#### 差异分解

将总差异分解为各驱动因素：

- **量/规模效应：** 按上期价格计算的量变化
- **价格/费率效应：** 按本期量计算的价格/费率变化
- **结构效应：** 不同价格/利润率项目之间的结构转变
- **新增/终止项目：** 仅在某一期间存在的项目
- **一次性/非循环项目：** 预计不会重复的项目
- **时间性影响：** 期间转移项目（非真实的运行率变化）
- **汇率影响：** 汇率变动对折算结果的影响

#### 调查与说明

对每项重大差异：
1. 量化差异（金额和百分比）
2. 判断是有利还是不利
3. 用上述类别分解驱动因素
4. 提供业务原因的叙述性说明
5. 评估差异是暂时性的还是代表趋势变化
6. 注明所需采取的行动（进一步调查、更新预测、流程变更）

## GAAP列报要求

### 利润表（ASC 220 / IAS 1）

- 列报期间内确认的所有收入和费用项目
- 费用按性质（原材料、人工、折旧）或职能（营业成本、研发、销售市场、一般行政）分类——美国公司通常按职能分类
- 按职能分类时，在附注中按性质披露折旧、摊销和员工福利费用
- 单独列报营业和非营业项目
- 将所得税费用单独列示为一行
- 美国GAAP和IFRS均禁止列报特殊项目
- 终止经营业务单独列报，税后净额

**常见列报注意事项：**

- **收入分拆：** ASC 606要求将收入分拆为能反映经济因素如何影响收入性质、金额、时间及不确定性的类别
- **股权激励：** 在职能费用类别内分类（研发、销售市场、一般行政），并在附注中披露股权激励总额
- **重组费用：** 如金额重大则单独列报，或纳入营业费用并在附注中披露
- **非GAAP调整：** 如列报非GAAP指标（盈利公告中常见），应清晰标注并与GAAP指标进行调节

### 资产负债表（ASC 210 / IAS 1）

- 区分流动资产/负债与非流动资产/负债
- 流动：预计在12个月内（或更长的经营周期内）变现、消耗或结清
- 资产按流动性顺序列报（流动性最强的在前）——美国标准做法
- 应收账款扣减信用损失准备列报（ASC 326）
- 固定资产扣减累计折旧列报
- 商誉不摊销——每年进行减值测试（ASC 350）
- 租赁：对经营性租赁和融资性租赁确认使用权资产和租赁负债（ASC 842）

### 现金流量表（ASC 230 / IAS 7）

- 间接法最为常见（从净利润开始，调整非现金项目）
- 直接法允许使用，但较少采用（需附补充间接法调节表）
- 必须披露已支付利息和已缴所得税（在表面或附注中）
- 非现金投融资活动单独披露（如租赁取得资产、以股票支付的并购对价）
- 现金等价物：原始期限不超过3个月的短期、高流动性投资

## 常见调整与重分类

### 期末调整

1. **应计：** 记录已发生但尚未支付的费用（应付账款应计、薪资应计、利息应计）
2. **递延：** 调整预付费用、递延收入和递延成本
3. **折旧与摊销：** 根据固定资产和无形资产明细表计提折旧/摊销
4. **坏账准备：** 根据账龄分析和历史损失率调整信用损失准备
5. **存货调整：** 对陈旧、滞销或减值存货进行减记
6. **外汇重估：** 按期末汇率重估外币货币性资产和负债
7. **税务拨备：** 记录当期和递延所得税费用
8. **公允价值调整：** 对投资、衍生工具及其他公允价值项目进行市价调整

### 重分类

1. **流动/非流动重分类：** 将12个月内到期的长期债务重分类至流动项目
2. **备抵科目抵销：** 将备抵与应收账款毛额、累计折旧与资产原值抵销列报
3. **内部往来抵消：** 在合并时抵消内部往来余额和交易
4. **终止经营业务：** 将终止经营业务的结果重分类至单独行项目
5. **权益法调整：** 记录权益法被投资方应分担的损益
6. **分部重分类：** 确保交易按经营分部正确分类
