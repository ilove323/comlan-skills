# 差异分析

**重要提示**：本技能协助完成差异分析工作流程，不提供财务建议。所有分析在用于报告前均应由具备资质的财务专业人员审核。

差异分解技术、重要性阈值、叙述性说明生成、瀑布图方法论，以及预算与实际与预测的对比分析。

## 差异分解技术

### 量价分解

最基本的差异分解方法。适用于收入、销售成本，以及任何可表达为价格×数量的指标。

**公式：**
```
总差异 = 实际 - 预算（或上期）

量效应 = （实际量 - 预算量）× 预算价格
价格效应 = （实际价格 - 预算价格）× 实际量
结构效应 = 残差（交互项），或按比例分配

验证：量效应 + 价格效应 = 总差异
      （当结构效应已嵌入量/价格效应中时）
```

**三因素分解（单独分离结构效应）：**
```
量效应 = （实际量 - 预算量）× 预算价格 × 预算结构
价格效应 = （实际价格 - 预算价格）× 预算量 × 实际结构
结构效应 = 预算价格 × 预算量 × （实际结构 - 预算结构）
```

**示例——收入差异：**
- 预算：10,000件 × $50 = $500,000
- 实际：11,000件 × $48 = $528,000
- 总差异：+$28,000（有利）
  - 量效应：+1,000件 × $50 = +$50,000（有利——销量增加）
  - 价格效应：-$2 × 11,000件 = -$22,000（不利——单价下降）
  - 净额：+$28,000

### 费率/结构分解

用于分析不同业务单元具有不同单位经济效益时的综合费率。

**公式：**
```
费率效应 = 各分项（实际量_i × （实际费率_i - 预算费率_i））之和
结构效应 = 各分项（预算费率_i × （实际量_i - 按预算结构计算的预计量_i））之和
```

**示例——毛利率差异：**
- 产品A：60%利润率，产品B：40%利润率
- 预算结构：50% A、50% B → 混合利润率50%
- 实际结构：40% A、60% B → 混合利润率48%
- 结构效应解释了2个百分点的利润率压缩

### 人员/薪酬分解

用于分析薪资和人员成本差异。

```
薪酬总差异 = 实际薪酬 - 预算薪酬

分解为：
1. 人头差异 = （实际人头 - 预算人头）× 预算平均薪酬
2. 费率差异 = （实际平均薪酬 - 预算平均薪酬）× 预算人头
3. 结构差异 = 因职级/部门结构变动导致的差异
4. 时间差异 = 早于/晚于计划招人（不完整期间效应）
5. 离职影响 = 非计划离职带来的节省（部分被填岗成本抵消）
```

### 支出类别分解

当量价分解不适用时，用于运营费用分析。

```
营业费用总差异 = 实际营业费用 - 预算营业费用

按以下类别分解：
1. 人员驱动成本     （工资、福利、工资税、招聘）
2. 量驱动成本       （主机托管、交易费、佣金、运输）
3. 酌情支出         （差旅、会议、专业服务、营销活动）
4. 合同/固定成本    （租金、保险、软件许可证、订阅）
5. 一次性/非循环    （离职补偿、诉讼和解、减记、项目费用）
6. 时间/节奏        （支出在期间之间的转移 vs. 计划）
```

## 重要性阈值和调查触发条件

### 设定阈值

重要性阈值决定哪些差异需要调查和叙述性说明。设定阈值应基于：

1. **财务报表重要性：** 通常为关键基准（收入、总资产、净利润）的1-5%
2. **行项目规模：** 较大行项目适用较低百分比阈值
3. **波动性：** 波动较大的行项目可能需要较高阈值以避免噪音干扰
4. **管理层关注度：** 何种程度的差异会影响决策？

### 建议阈值框架

| 对比类型 | 金额阈值 | 百分比阈值 | 触发条件 |
|---------|---------|-----------|---------|
| 实际 vs. 预算 | 按组织设定 | 10% | 任一超过即触发 |
| 实际 vs. 上期 | 按组织设定 | 15% | 任一超过即触发 |
| 实际 vs. 预测 | 按组织设定 | 5% | 任一超过即触发 |
| 环比（月度） | 按组织设定 | 20% | 任一超过即触发 |

*金额阈值应根据组织规模设定。常见做法：利润表项目为收入的0.5%-1%。*

### 调查优先级

当多项差异超过阈值时，按以下顺序优先调查：

1. **绝对金额最大** ——利润表影响最大
2. **百分比最大** ——可能表明流程问题或错误
3. **方向出乎意料** ——差异方向与趋势或预期相反
4. **新出现的差异** ——此前走势正常，现在出现偏差
5. **累积/持续扩大** ——每期均在扩大

## 差异说明叙述生成

### 每项差异说明的结构

```
[行项目]：[有利/不利]差异 $[金额]（[百分比]%）
vs. [对比基准] for [期间]

驱动因素：[主要驱动因素描述]
[2-3句话解释差异的业务原因，并量化各驱动因素的贡献]

展望：[一次性 / 预计持续 / 改善中 / 恶化中]
行动：[无需处理 / 监控 / 进一步调查 / 更新预测]
```

### 差异说明质量检查清单

好的差异说明应：

- [ ] **具体：** 点名实际驱动因素，而非"高于预期"
- [ ] **量化：** 包含每项驱动因素的金额和百分比影响
- [ ] **有因果关系：** 解释"为何"发生，而非仅说明"发生了什么"
- [ ] **前瞻性：** 说明差异是否预计持续
- [ ] **可操作：** 指出所需的后续跟进或决策
- [ ] **简洁：** 2-4句话，而非大段废话

### 常见差异说明反模式（应避免）

- "收入高于预算是因为收入较高"（循环说法——没有实质解释）
- "本期费用有所上升"（模糊——哪些费用？为什么？）
- 说"时间性影响"但未说明早/晚了什么及何时恢复正常
- 说"一次性"但未解释具体是什么项目
- 对重大差异笼统归因于"各类小项"（必须进一步分解）
- 只聚焦最大驱动因素而忽略抵消项目

## 瀑布图方法论

### 概念

瀑布图（或桥接图）展示如何通过一系列正负贡献从一个数值演变至另一个数值，用于直观呈现差异分解。

### 数据结构

```
起始值：[基准/预算/上期金额]
驱动因素：[各贡献因素及其带符号金额]
结束值：[实际/本期金额]

验证：起始值 + 所有驱动因素之和 = 结束值
```

### 文字瀑布图格式

当无法使用图表工具时，以文字瀑布图呈现：

```
瀑布图：收入 — Q4实际 vs. Q4预算

Q4预算收入                                    $10,000K
  |
  |--[+] 销量增长（新客户）                    +$800K
  |--[+] 扩展收入（存量客户）                  +$400K
  |--[-] 降价/折扣                             -$200K
  |--[-] 客户流失/萎缩                         -$350K
  |--[+] 外汇顺风                              +$50K
  |--[-] 时间性影响（订单滑至Q1）              -$150K
  |
Q4实际收入                                    $10,550K

净差异：+$550K（+5.5%，有利）
```

### 桥接调节表

用调节表补充瀑布图：

| 驱动因素 | 金额 | 占总差异% | 累计 |
|---------|------|---------|------|
| 销量增长 | +$800K | 145% | +$800K |
| 扩展收入 | +$400K | 73% | +$1,200K |
| 降价 | -$200K | -36% | +$1,000K |
| 客户流失 | -$350K | -64% | +$650K |
| 外汇顺风 | +$50K | 9% | +$700K |
| 时间性（订单滑落） | -$150K | -27% | +$550K |
| **总差异** | **+$550K** | **100%** | |

*注：当存在相互抵消项时，单项驱动因素的百分比可能超过100%。*

### 瀑布图最佳实践

1. 从最大正项到最大负项排列（或按业务逻辑顺序）
2. 最多保留5-8个驱动因素——小项目合并为"其他"
3. 验证瀑布图调节正确（起始值 + 驱动因素 = 结束值）
4. 颜色编码：绿色表示有利，红色表示不利（可视化图表中）
5. 每个柱状图标注金额和简短描述
6. 包含"总差异"汇总柱

## 预算 vs. 实际 vs. 预测对比

### 三向对比框架

| 指标 | 预算 | 预测 | 实际 | 预算差异（金额） | 预算差异（%） | 预测差异（金额） | 预测差异（%） |
|------|------|------|------|-------------|-----------|-------------|-----------|
| 收入 | $X | $X | $X | $X | X% | $X | X% |
| 营业成本 | $X | $X | $X | $X | X% | $X | X% |
| 毛利润 | $X | $X | $X | $X | X% | $X | X% |

### 各对比类型的适用场景

- **实际 vs. 预算：** 全年绩效衡量、薪酬决策、董事会报告。预算在年初制定，通常不予更改。
- **实际 vs. 预测：** 运营管理、识别新出现的问题。预测定期更新（每月或每季度）以反映当前预期。
- **预测 vs. 预算：** 了解预期相对于规划如何变化，有助于识别预测准确性问题。
- **实际 vs. 上期：** 趋势分析、环比表现，在预算意义不大时尤为有用（新业务线、并购后）。
- **实际 vs. 去年同期：** 同比增长分析、经季节性调整的对比。

### 预测准确性分析

跟踪预测准确性以改进规划：

```
预测准确性 = 1 - |实际 - 预测| / |实际|

MAPE（平均绝对百分比误差）= 各期 |实际 - 预测| / |实际| 的平均值
```

| 期间 | 预测 | 实际 | 差异 | 准确性 |
|------|------|------|------|-------|
| 1月 | $X | $X | $X（X%） | XX% |
| 2月 | $X | $X | $X（X%） | XX% |
| ... | ... | ... | ... | ... |
| **平均** | | | **MAPE** | **XX%** |

### 差异趋势分析

跟踪差异在全年的演变，识别系统性偏差：

- **持续有利：** 预算可能过于保守（压低目标）
- **持续不利：** 预算可能过于激进，或存在执行问题
- **不利差距扩大：** 业绩恶化或目标不切实际
- **差距收窄：** 预测准确性通过全年改善（正常模式）
- **波动较大：** 业务不可预测或预测方法论有缺陷
