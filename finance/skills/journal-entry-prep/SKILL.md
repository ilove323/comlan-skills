# 日记账凭证编制

**重要提示**：本技能协助完成日记账凭证工作流程，不提供财务建议。所有凭证在过账前均应由具备资质的财务专业人员审核。

日记账凭证编制的最佳实践、标准凭证类型、文档要求及审核工作流程。

## 标准应计类型及其凭证

### 应付账款应计

对期末已收货或服务但尚未开票的项目进行应计。

**典型凭证：**
- 借：费用科目（或符合资本化条件的资产科目）
- 贷：应计负债

**计算依据来源：**
- 已确认收货的未结采购订单
- 已提供服务但尚未开票的合同
- 循环性供应商安排（水电、订阅、专业服务）
- 已提交但尚未处理的员工报销申请

**关键注意事项：**
- 次期自动冲回（建议设置自动冲回）
- 各期使用一致的估算方法
- 记录估算依据（采购订单金额、合同条款、历史运行率）
- 跟踪实际金额与应计金额以优化未来估算

### 固定资产折旧

对有形和无形资产计提期间折旧费用。

**典型凭证：**
- 借：折旧/摊销费用（按部门或成本中心）
- 贷：累计折旧/摊销

**折旧方法：**
- **直线法：**（原值 - 残值）/ 使用年限——财务报告最常用
- **加速折旧法：** 对账面净值按固定比率计提的加速折旧方法
- **产量法：** 基于实际使用量或产出量与预计总量之比

**关键注意事项：**
- 从固定资产台账或明细表计提折旧
- 核实新增资产已正确设置使用年限和折旧方法
- 检查需要核销的资产处置或减值
- 确保账面折旧与税务折旧追踪的一致性

### 预付费用摊销

在受益期间内摊销预付费用。

**典型凭证：**
- 借：费用科目（保险、软件、租金等）
- 贷：预付费用

**常见预付类别：**
- 保险费（通常为12个月期保单）
- 软件许可证和订阅费
- 预付租金（如适用于租赁条款）
- 预付维保合同
- 会议和活动定金

**关键注意事项：**
- 维护含起止日期和月摊销额的摊销明细表
- 检查是否有应直接费用化的预付项（金额不重要的）
- 核查已取消或终止的合同是否需要加速摊销
- 确保新增预付项及时录入明细表

### 薪资应计

对期间应计的薪酬及相关费用进行应计。

**典型凭证：**

*工资应计（工资期间不与月末对齐时）：*
- 借：工资费用（按部门）
- 贷：应计工资

*奖金应计：*
- 借：奖金费用（按部门）
- 贷：应计奖金

*福利应计：*
- 借：福利费用
- 贷：应计福利

*工资税应计：*
- 借：工资税费用
- 贷：应计工资税

**关键注意事项：**
- 根据期间工作日与工资期间之比计算工资应计
- 奖金应计应反映计划条款（目标金额、绩效指标、支付时间）
- 包括雇主方税款和福利（社会保险、失业保险、医疗、养老金配捐）
- 如政策或法规要求，追踪带薪假/年假应计负债

### 收入确认

根据履约义务和交付情况确认收入。

**典型凭证：**

*确认之前递延的收入：*
- 借：递延收入
- 贷：收入

*确认收入并产生应收款：*
- 借：应收账款
- 贷：收入

*预收款项递延：*
- 借：现金 / 应收账款
- 贷：递延收入

**关键注意事项：**
- 遵循ASC 606五步法处理客户合同
- 识别每份合同中的独立履约义务
- 确定交易价格（含可变对价）
- 将交易价格分配给各履约义务
- 在履约义务履行时/履行后确认收入
- 保留合同层面明细以支持审计

## 支持文件要求

每份日记账凭证应具备：

1. **凭证描述/摘要：** 清晰具体地描述凭证所记录的内容及原因
2. **计算支持：** 金额的推导过程（公式、明细表、源数据引用）
3. **源凭证：** 底层交易或事项的引用（采购订单号、发票号、合同引用、工资明细）
4. **期间：** 凭证适用的会计期间
5. **编制人识别：** 编制人及编制时间
6. **审批：** 按授权矩阵的复核和审批证明
7. **冲回标志：** 凭证是否自动冲回及冲回日期

## 复核与审批工作流程

### 典型审批矩阵

| 凭证类型 | 金额阈值 | 审批人 |
|---------|---------|-------|
| 标准循环类 | 任何金额 | 会计主管 |
| 非循环/手工类 | < 5万元 | 会计主管 |
| 非循环/手工类 | 5万 - 25万元 | 主计长 |
| 非循环/手工类 | > 25万元 | CFO/财务副总裁 |
| 顶层/合并类 | 任何金额 | 主计长或以上 |
| 跨期调整类 | 任何金额 | 主计长或以上 |

*注：阈值应根据组织的重要性水平和风险容忍度设定。*

### 复核清单

审批日记账凭证前，复核人应核实：

- [ ] 借方等于贷方（凭证平衡）
- [ ] 会计期间正确（未过账至已关闭期间）
- [ ] 科目代码存在且适用于该交易
- [ ] 金额计算准确并有计算支持
- [ ] 描述清晰、具体，足以用于审计
- [ ] 部门/成本中心/项目编码正确
- [ ] 处理方式与上期及会计政策一致
- [ ] 自动冲回设置适当（应计类凭证应冲回）
- [ ] 支持文件齐全并有引用
- [ ] 凭证金额在编制人的权限范围内
- [ ] 无与现有凭证重复
- [ ] 异常或大额金额已有解释说明

## 常见错误检查

1. **凭证不平衡：** 借方不等于贷方（系统应能防止，但手工凭证须检查）
2. **期间错误：** 凭证过账至错误或已关闭的期间
3. **符号错误：** 借方误录为贷方或反之
4. **重复凭证：** 同一交易重复记录（过账前检查重复）
5. **科目错误：** 凭证过账至错误总账科目（尤其是相似科目代码）
6. **遗漏冲回：** 应计凭证未设置自动冲回，导致重复计算
7. **陈旧应计：** 循环应计未随情况变化而更新
8. **整数估算：** 可疑的整数金额，可能未反映实际计算
9. **外汇汇率错误：** 外币凭证使用了错误汇率或日期
10. **遗漏内部往来抵消：** 实体间分录没有相应的抵消分录
11. **资本化错误：** 应资本化的费用或应费用化的资本项目
12. **截止错误：** 根据交付或服务日期应记录在不同期间的交易
