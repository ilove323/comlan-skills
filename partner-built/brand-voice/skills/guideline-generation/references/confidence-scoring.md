# 置信度评分方法

如何为生成的品牌指南分配和解释置信度评分。

## 评分级别

### 高置信度
指南部分得到充分支持且可操作。

**标准（至少满足 3 项）：**
- 3 个以上相互印证的来源
- 在至少一个权威来源中找到明确指导
- 文档分析和对话分析结果一致
- 具体、可操作的说明（不仅仅是模糊原则）
- 无未解决的冲突

**示例：**声音属性"自信但不傲慢"出现在官方风格指南中，在邮件模板中有所体现，并与顶级绩效者通话中的模式相符。

### 中置信度
该部分合理，但可以从更多数据或团队确认中受益。

**标准（至少满足 2 项）：**
- 1-2 个印证来源
- 从模式推断而非明确说明
- 通过时效性或权威性解决的小冲突
- 可操作但需要一定解读
- 可能存在一个未解决的冲突

**示例：**社交媒体语气从邮件模板和一个 Slack 讨论串推断，但未找到官方社交媒体指南。

### 低置信度
该部分是尽力而为的建议。强烈建议团队审查。

**标准（至少满足 2 项）：**
- 仅单一来源
- 主要从间接证据推断
- 需要大量解读
- 来源间存在未解决的冲突
- 具体性有限

**示例：**竞争定位来自讨论竞争对手的单一销售通话，没有支持文档。

## 部分级别评分指南

### 声音属性
- **高**：属性出现在官方品牌指南中，并在模板或通话中有所体现
- **中**：属性仅出现在一种文档类型中，或从多次对话中推断
- **低**：属性从单一来源推断，或从间接证据推断

### 消息框架
- **高**：价值主张在官方素材中记录，且在销售对话中持续使用
- **中**：有记录但未在实践中观察到，或在实践中观察到但未记录
- **低**：从单一 PPT 或单一通话中提取

### 语气矩阵
- **高**：该情境存在明确的语气指导，且与观察到的行为相符
- **中**：从该情境中 3 个以上内容示例推断语气
- **低**：从 1-2 个示例推断，或从类似情境外推

### 术语
- **高**：术语在风格指南或词汇表中明确列出
- **中**：术语在模板和通话中持续使用（基于模式）
- **低**：术语在单一文档中观察到，或从品牌个性推断

### 语言模式（来自录音）
- **高**：在 5 个以上跨多位发言者的通话中观察到模式
- **中**：在 3-4 个通话中观察到，或来自单一顶级绩效者
- **低**：仅在 1-2 个通话中观察到

### 以录音为主的场景

当指南主要从对话来源生成（无权威文档可用）：
- 从 5 个以上录音推导的声音属性 = **中**（而非低）
- 从跨 5 个以上通话的一致模式得出的消息框架 = **中**
- 在总体计算中，语言模式权重从 10% 增加到 20%（从声音属性中减去 10%）

在指南元数据中注明："主要从对话来源生成指南 — 建议团队审查以正式确认。"

## 总体置信度

将各部分评分的加权平均值计算为总体指南置信度：

| 部分 | 权重 |
|------|------|
| 声音属性 | 30% |
| 消息框架 | 25% |
| 语气矩阵 | 20% |
| 术语 | 15% |
| 语言模式 | 10% |

评分转换：高 = 1.0，中 = 0.6，低 = 0.3

**示例：**
- 声音属性：高（1.0 x 0.30 = 0.30）
- 消息：中（0.6 x 0.25 = 0.15）
- 语气：中（0.6 x 0.20 = 0.12）
- 术语：高（1.0 x 0.15 = 0.15）
- 语言：低（0.3 x 0.10 = 0.03）
- **总体：0.75 = 中-高置信度**

**总体评分阈值：**
- 0.85–1.0 = 高
- 0.60–0.84 = 中
- 低于 0.60 = 低

## 呈现方式

在每个部分标题旁展示置信度：

```markdown
## 声音属性（置信度：高）
[内容]

## 语气矩阵（置信度：中）
[内容 — 注：未找到官方社交媒体指南，语气从邮件模式推断]
```

对于中和低置信度部分，包含简短说明，解释为何置信度有限以及什么会提升它。

## 与开放性问题的关系

低置信度部分应生成对应的开放性问题：

- **低置信度 + 冲突** = 高优先级开放性问题
- **低置信度 + 缺口** = 中优先级开放性问题
- **中置信度 + 小冲突** = 低优先级开放性问题

每个开放性问题都包含一个建议，如果得到确认，将提升该部分的置信度评分。
