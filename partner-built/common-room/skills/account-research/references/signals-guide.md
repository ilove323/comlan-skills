# Common Room 信号类型 — 解释指南

## 信号类别

### 产品信号（第一方）

| 信号 | 含义 |
|------|------|
| 活跃席位增加 | 账户内扩展 — 外联机会 |
| 功能采用（使用了新功能） | 产品参与度加深 — 适合扩展对话 |
| 登录频率激增 | 参与度提升 — 跟进了解原因 |
| 登录频率下降 | 有脱离或流失的风险 — 需要主动外联 |
| 试用已开始 | 正在积极评估 — 时间敏感，优先处理 |
| 试用即将到期 | 决策点即将到来 — 立即行动 |
| 已开启支持工单 | 潜在摩擦 — 检查是否有规律 |

### 社区信号（第二方）

| 信号 | 含义 |
|------|------|
| 新社区帖子 | 积极参与，有问题或想法 |
| 回复/对帖子的反应 | 社区参与 |
| 发起论坛讨论串 | 在寻求帮助或分享反馈 — 可以直接参与 |
| 已查看文档（特定页面） | 在研究特定功能 — 如果与扩展领域相关，价值很高 |
| 活动注册/参加 | 强烈的参与和意图信号 |

### 意图信号（第三方）

| 信号 | 含义 |
|------|------|
| 为相关职位招聘 | 团队在扩张，有预算 |
| 宣布融资 | 很可能有新预算 — 发送祝贺外联 |
| 公司新闻/媒体提及 | 外联的好钩子 |
| 技术栈变化 | 可能需要新的集成 |
| 竞争产品从技术栈中移除 | 机会窗口 — 非常高优先级 |
| 网站活动激增 | 处于积极研究阶段 |

### 关系信号

| 信号 | 含义 |
|------|------|
| 已记录近期会议 | 关系活跃 — 有背景可用 |
| 超过 90 天没有会议 | 关系趋冷 — 请多关注 |
| CRM 中有开放的机会 | 有活跃交易 — 与客户经理协调 |
| 冠军工作变动 | 关系风险或机会 |

## 信号解读原则

1. **时效性最重要。**昨天的信号远比 6 个月前的重要。始终注明时间戳。
2. **信号群集 > 单一信号。**多个同方向的信号比任何单一信号都强得多。
3. **信号缺失本身也是信号。**付费账户中最少的使用 + 没有社区参与 = 风险指标。
4. **相对于账户阶段进行背景化解读。**试用期：转化信号为主。成熟客户：扩展和流失风险为主。潜在客户：意图信号最重要。
