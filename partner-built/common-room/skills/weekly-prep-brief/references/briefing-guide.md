# 每周准备简报 — 结构和边缘情况

## 简报深度指南

根据账户类型校准研究深度：

| 账户类型 | 深度 |
|---------|------|
| 战略账户（大型、高 ARR） | 完整 — 账户 + 所有参与者 |
| 活跃机会（开放的交易） | 完整 — 强调交易信号 |
| 90 天内续约 | 完整 — 强调风险/健康信号 |
| 中市场客户，例行通话 | 标准 — 简化账户 + 关键参与者 |
| 小账户，快速跟进 | 最简 — 3-4 个要点，单一参与者 |
| 全新潜在客户（第一次会议） | 完整联系人 + 账户 — 第一印象很重要 |

## 本周概述优先事项

在简报顶部突出标注以下情况：

- 7 天内试用到期、30 天内续约，或检测到流失风险信号
- 开放的关键支持工单或未解决的升级
- 值得在通话中提出的扩展信号
- 参加的高管利益相关者或新冠军
- 冠军近期换工作或账户中出现竞争信号
- 近期公司融资、产品里程碑或强烈的社区倡导（用作开场话题）

## 边缘情况

- **没有 Common Room 数据：**明确说明，进行外部网络研究，建议代表向团队了解线下背景。
- **与同一公司的多次会议：**研究账户一次，跨会议共享，通过参与者和目标区分。
- **一周内 7 次以上会议：**询问是优先对特定账户进行完整研究，还是对所有账户进行简化准备；默认对战略账户进行完整研究。
- **当天会议：**跳过确认，生成压缩版本（公司快照 + 顶级信号 + 1-2 个话语点），标记为"[快速准备]"。
- **取消的会议：**默默排除；如果状态不明确，包含说明"[状态不明确 — 请确认此会议是否仍然进行。]"
