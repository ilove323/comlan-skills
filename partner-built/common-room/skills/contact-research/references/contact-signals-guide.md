# 联系人信号 — 解释指南

## 联系人信号类型

### 活动信号

| 信号 | 解释 |
|------|------|
| 邮件打开/回复 | 正在积极关注您的组织；回复 = 更高意图 |
| 参加会议 | 关系已建立；注意会议类型（演示、QBR 等） |
| 社区帖子或回复 | 有问题、问题或想法 — 高参与信号 |
| 开启支持工单 | 遇到摩擦 — 同理心机会 |
| 访问了文档页面 | 在研究特定领域 — 注意是哪些页面 |
| 产品登录 | 活跃用户 — 检查频率和时效性 |
| 活动注册/参加 | 强烈的参与和意图信号 |

### 关系信号

| 信号 | 解释 |
|------|------|
| 超过 30 天无活动 | 关系趋冷 — 适合温和地回访 |
| 超过 90 天无活动 | 沉默 — 可能需要从不同角度重新激活 |
| 跨多个渠道持续活跃 | 冠军或高度参与的用户 — 高价值联系人 |
| 沉默后活动激增 | 有什么变化了 — 调查原因 |

### 工作/职业信号

| 信号 | 解释 |
|------|------|
| 近期工作变动（新公司） | 在新公司的热情介绍机会；旧公司的关系可能发生变化 |
| 在同一公司晋升 | 影响力增长 — 关系变得更有价值 |
| 职位变为决策者角色 | 升级关系策略 |
| 公司扩张（其团队有新员工） | 可能有预算；扩展机会 |

## Spark 角色分类

- **冠军** — 每日用户，在内部倡导
- **经济买家** — 掌握预算权，可能不直接使用产品
- **技术评估者** — 评估匹配性和集成，影响技术决策
- **终端用户** — 主要产品用户，通过 NPS/反馈影响续约
- **把关人** — 控制访问权，需要谨慎应对

## 对话开场白

一个强有力的开场白要引用具体内容，展示您做了功课，在不强迫的情况下打开一扇门，并与价值相连。

- "我看到了您在社区关于 [话题] 的问题 — 我们刚发布了一个功能，正好解决了这个问题。介绍一下是否有用？"
- "您团队上个月对 [功能] 的使用量猛增 — 想知道您是否在那个阶段遇到了 [常见障碍]。"
