# Common Room 插件

由 Common Room 驱动的 GTM 工作流 — 账户研究、联系人研究、通话准备、个性化外联、潜客挖掘和每周简报。

## 概述

此插件将 AI助手 连接到 Common Room 的 MCP 服务器，并配备了六项技能，覆盖最常见的销售代表工作流。所有输出均以 Common Room 真实信号数据为基础 — 第一方产品信号、第二方社区信号、第三方意图信号，以及来自 RoomieAI 和 Spark 的丰富信息。

## 前提条件

- **Common Room MCP**（`mcp.commonroom.io/mcp`）必须已连接并完成认证。这是所有插件功能的主要数据来源。
- **日历连接器**（可选）— 在 `call-prep` 和 `weekly-prep-brief` 中启用自动会议查找。如果未连接，两个技能将改为向用户询问会议详情。

## 技能

技能通过对话触发。描述您想做的事，AI助手 会自动加载合适的技能。

| 技能 | 触发短语 |
|------|----------|
| `account-research` | "研究 [公司]"、"告诉我关于 [域名] 的信息"、"[账户] 最近怎样"、"[公司] 是否显示购买信号" |
| `contact-research` | "[姓名] 是谁"、"查找 [邮箱]"、"研究 [联系人]"、"[姓名] 是热门线索吗" |
| `call-prep` | "帮我准备与 [公司] 的通话"、"为与 [公司] 的会议做准备"、"在与 [公司] 交谈之前我应该了解什么" |
| `compose-outreach` | "起草给 [人员] 的外联"、"给 [姓名] 写一封邮件"、"为 [联系人] 撰写一条消息" |
| `prospect` | "找到符合 [条件] 的公司"、"建立潜客列表"、"找到 [类型公司] 的联系人" |
| `weekly-prep-brief` | "每周准备简报"、"准备我的一周"、"这周我有哪些通话" |

## 命令

两个命令用于受益于明确调用的复杂工作流：

| 命令 | 用法 |
|------|------|
| `/generate-account-plan <公司>` | 包含利益相关者映射、参与分析、机会、风险和行动项的全面战略账户计划 |
| `/weekly-brief [日期范围]` | 生成完整的每周准备简报（默认为未来 7 天） |

## 每项技能的产出

**账户研究** — 处理四种模式：完整概览、目标字段问题、诚实的稀疏数据响应，以及 MCP 数据 + LLM 推理的组合。包含最新新闻的网络搜索。自动限定范围到"我的细分市场"。

**联系人研究** — 通过邮件、姓名+公司或社交账号进行查找。返回丰富的身份信息、CRM 字段、评分、网站访问、活动历史、Spark 分析和对话开场白。

**通话准备** — 公司快照、每位参与者档案、信号亮点、针对性话语点、可能的异议和推荐的通话目标。优先处理 Gong/通话录音活动。如果已连接，支持日历感知。

**撰写外联** — 三种个性化格式（邮件、通话脚本、LinkedIn 消息），以 Common Room 信号和网络搜索钩子为基础。在可用时，针对用户公司的定位进行定制。

**潜客挖掘** — 区分净新公司（ProspectorOrganization）和现有账户（Organization）。支持迭代细化和相似查找（"找到像 [X] 这样的公司"）。网络搜索丰富净新结果。

**每周准备简报** — 覆盖未来 7 天每个外部通话的完整简报：公司快照、参与者档案、信号和每次会议的推荐目标。

## 设置

1. 确保 Common Room MCP 服务器在您的 OpenClaw 设置中已连接并完成认证。
2. （可选）连接日历 MCP 服务器，以便在通话准备和每周简报中自动查找会议。
3. 安装此插件。所有技能和命令立即可用。

## 用户上下文

所有限定用户领域范围的技能都会自动从 Common Room 获取 `Me` 对象。这提供了用户的档案、角色和"我的细分市场" — 确保查询默认限定在他们的领域范围内。详见 `references/me-context.md`。

当公司上下文可用时，技能会根据用户的产品和 ICP 量身定制建议。详见 `references/my-company-context.md`。
