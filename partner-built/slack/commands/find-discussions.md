---
name: find-discussions
description: 跨 Slack 频道查找关于特定话题的讨论。触发词：/find-discussions、查找讨论
user-invocable: true
---

根据 $ARGUMENTS 中提供的话题：

1. 使用 `slack_search_public` 工具搜索与话题匹配的消息。首先将话题作为自然语言问题使用，以获得语义搜索结果。
2. 如果语义搜索结果稀少，使用话题中的关键词进行关键词搜索跟进。
3. 对于最相关的结果，使用 `slack_read_thread` 获取完整的讨论串对话，以捕获完整的讨论背景。
4. 按相关性组织呈现结果：
   - 对于找到的每个讨论，显示：频道名称、发起者、对话的简短摘要以及日期。
   - 如果讨论跨多个频道，将相关讨论分组在一起。
   - 突出显示任何结论、决策或未解决的问题。
5. 将输出限制在最多 5-10 个最相关的讨论，以保持结果可管理。
6. 如果没有找到结果，建议用户可以尝试的替代搜索词或更广泛的查询。
