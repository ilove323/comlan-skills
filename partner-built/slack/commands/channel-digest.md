---
name: channel-digest
description: 获取多个 Slack 频道的最近活动摘要。触发词：/channel-digest、频道摘要、获取频道动态
user-invocable: true
---

根据 $ARGUMENTS 中提供的逗号分隔的频道名称（去除每个名称开头的 `#` 和空白）：

1. 将参数解析为各个频道名称。去除每个名称开头的 `#` 和空白。

2. 对于每个频道：
   a. 使用 `slack_search_channels` 查找频道 ID。
   b. 使用 `slack_read_channel` 读取最近消息（每个频道使用最多 50 条消息的限制以保持可管理性）。
   c. 总结该频道的关键活动：主要话题、决策、问题和值得关注的消息。

3. 按以下格式呈现摘要：

   ```
   *频道摘要 — <今日日期>*

   *#频道-1*
   - 摘要要点 1
   - 摘要要点 2

   *#频道-2*
   - 摘要要点 1
   - 摘要要点 2

   ...
   ```

4. 对于每个频道，将摘要控制在最多 3-5 个要点。专注于可操作或值得关注的内容。

5. 如果某个频道最近没有活动，说明它最近很安静，并提及（如果可见）上次发布消息的时间。

6. 如果找不到某个频道名称，告知用户并继续处理其余频道。
