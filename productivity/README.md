# 效率工具插件

这是一款主要为 [OpenClaw](https://openclaw.ai) 设计的效率插件——同时也适用于 Claude Code。任务管理、工作记忆和可视化仪表板——AI助手学习您的人员、项目和术语，从而像同事而非聊天机器人一样行事。

## 安装

```
openclaw skills install productivity
```

## 功能

此插件让 AI助手对您的工作有持续性的理解：

- **任务管理** — 一个 AI助手可读写并执行的 Markdown 任务列表（`TASKS.md`）。自然地添加任务，AI助手会跟踪状态、处理过时事项并与外部工具同步。
- **工作记忆** — 一个两层记忆系统，教 AI助手理解您的缩写、人员、项目和术语。说"让 todd 为 oracle 做 PSR"，AI助手就能确切知道是谁、做什么以及哪个项目。
- **可视化仪表板** — 一个本地 HTML 文件，提供任务的看板视图和 AI助手对您工作场所了解情况的实时视图。从看板或文件编辑——两者保持同步。

## 命令

| 命令 | 功能 |
|---------|--------------|
| `/start` | 初始化任务和记忆，打开仪表板 |
| `/update` | 处理过时事项、检查记忆缺口、从外部工具同步（如适用） |
| `/update --comprehensive` | 深度扫描邮件、日历、聊天——标记遗漏的待办事项并建议新记忆 |

## 技能

| 技能 | 描述 |
|-------|-------------|
| `memory-management` | 两层记忆系统——CLAUDE.md 用于工作记忆，memory/ 目录用于深度存储 |
| `task-management` | 使用共享 TASKS.md 文件进行基于 Markdown 的任务跟踪 |

## 示例工作流

### 入门

```
您：/start

AI助手：[创建 TASKS.md、CLAUDE.md、memory/ 目录和 dashboard.html]
        [在浏览器中打开仪表板]
        [询问您的角色、团队和当前优先事项以初始化记忆]
```

### 自然地添加任务

```
您：我需要在周五前审查 Sarah 的预算提案，
     与 Greg 同步后起草 Q2 路线图，并跟进
     Platform 团队的 API 规格

AI助手：[将所有三个任务添加到 TASKS.md 并附上背景]
        [仪表板自动更新]
```

### 晨间同步

```
您：/update --comprehensive

AI助手：[扫描邮件、日历和聊天中的新行动项]
        [标记："预算提案审查明天到期——仍未处理"]
        [建议："3 个线程中提到了新人：Jamie Park，
         设计负责人——添加到记忆？"]
        [更新过时任务并填补记忆缺口]
```

### 工作场所缩写

记忆填充后，AI助手能立即解码您的缩写：

```
您：让 todd 为 oracle 做 PSR

AI助手："请 Todd Martinez（财务负责人）为 Oracle Systems 项目
         （230 万美元，Q2 结束）准备流水线状态报告"
```

无需澄清问题。无需来回确认。

## 数据源

> 如果您看到不熟悉的占位符或需要检查已连接的工具，请参阅 [CONNECTORS.md](CONNECTORS.md)。

连接您的通信和项目管理工具以获得最佳体验。没有这些工具，可以手动管理任务和记忆。

**内置 MCP 连接：**
- 聊天（Slack）用于团队上下文和消息扫描
- 电子邮件和日历（Microsoft 365）用于行动项发现
- 知识库（Notion）用于参考文档
- 项目追踪器（Asana、Linear、Atlassian、monday.com、ClickUp）用于任务同步
- 办公套件（Microsoft 365）用于文档

**其他选项：**
- 参阅 [CONNECTORS.md](CONNECTORS.md) 了解每个类别的替代工具
