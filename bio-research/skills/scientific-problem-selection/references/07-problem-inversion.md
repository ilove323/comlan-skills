# 技能7：问题反转策略（"将其颠倒"）

## 概述
本技能提供三种具体策略，通过重新框架问题来绕过障碍。当卡住时，不要在当前方法上更加努力，而是尝试反转问题。

## 核心原则
**"将问题颠倒"**

三种有力的策略：
1. **解固定参数**（在技能4中已涵盖，在危机中应用）
2. **无法实现目标A？实现可比较的目标B**
3. **"我有答案；问题是什么？"**

## 策略1：解固定参数（在危机模式下）

**何时使用：** 项目执行中的常规问题

**方法：** 让一个"神圣"的固定参数浮动

**案例：**
- **卡住：** 肿瘤微环境中APC-T细胞相互作用的空间转录组学
- **全部固定：** 技术、细胞类型、背景
- **反转：**
  - 解固定技术 → 还有什么可以测量这些相互作用？
  - 解固定细胞类型 → 肿瘤中还有哪些其他相互作用重要？
  - 解固定背景 → APC-T相互作用在哪里还重要？

**你的应用：**
对于项目中的每个固定参数：
- 如果这个浮动了会怎样？
- 存在哪些替代方案？
- 哪个会更容易/更快/更有信息量？

## 策略2：可比较目标替换

**何时使用：** 项目面临存在性威胁（无法实现原始目标）

**方法：** 实现一个不同但同等有价值的目标

**心态转变：**
- **错误：** "我未能做到X"
- **正确：** "世界需要的是Y，而我可以做到Y"

**案例：PROTAC故事**
- **目标A（失败）：** 降解特定治疗靶点
- **目标B（实现）：** 绘制哪些激酶是可降解的
- **价值：** B更有影响力（普遍原则+方法验证）
- **学习：** 靶点结合≠降解（重要发现）

**框架：**
1. **原始目标：** [你想要的]
2. **为什么失败：** [具体原因]
3. **你用当前数据/工具能做什么：** [能力]
4. **可比较目标：**
   - 选项1：[不同但相关的目标]
   - 选项2：[另一个替代方案]
   - 选项3：[再一个]
5. **哪个最有价值：** [分析]
6. **如何框架：** [沟通策略]

## 策略3：答案寻找问题

**何时使用：** 项目末期挑战（解读、框架、应用）

**方法：** 你得到了一个答案，但不是针对你原始问题的。你的数据回答了什么问题？

**心态转变：**
- **错误：** "这没有回答我的问题"
- **正确：** "这回答了什么有趣的问题？"

**案例：类固醇受体**
- **原始问题：** 这种类固醇的受体是什么？
- **获得的答案：** 以不同亲和力结合多种受体
- **问题：** 无法回答原始问题（没有单一受体）
- **反转：** "这回答了什么问题？"
- **新问题：** 有限的受体库如何感知无限的脂质？
- **答案：** 组合感知（图案=独特的"和弦"）
- **影响：** 比预期发现更有趣

**框架：**
1. **原始问题：** [你问的]
2. **获得的数据：** [你实际发现的]
3. **为什么不回答：** [不匹配之处]
4. **数据清晰显示了什么：** [可靠发现]
5. **这些可能回答什么问题：**
   - 问题1：[选项]
   - 问题2：[选项]
   - 问题3：[选项]
6. **哪个最有趣：** [评估]
7. **如何重新框架论文/项目：** [新框架]

## 工作流程

### 第一阶段：识别你的障碍
- **类型：** 技术/生物学/竞争/解读
- **严重性：** 常规/存在性/末期
- **描述：** [什么在阻碍你]

### 第二阶段：选择策略

| 障碍类型 | 推荐策略 |
|---------|---------|
| 技术障碍，项目中期 | 策略1（解固定参数） |
| 无法实现原始目标 | 策略2（可比较目标） |
| 有数据，但不清楚含义 | 策略3（答案寻找问题） |

### 第三阶段：应用策略

使用你具体情况，完成上述相关框架。

### 第四阶段：评估替代方案

对于每个产生的替代方案：
- **科学价值：** 这有多有趣？
- **可行性：** 执行有多难？
- **时间表：** 需要多长时间？
- **影响：** 与原始计划相比如何？
- **你的优势：** 你在这里还有优势吗？
