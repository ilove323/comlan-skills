# 创建销售资料 — 快速参考

## 调用方式
```
/create-an-asset
/create-an-asset [CompanyName]
"Create an asset for [Company]"
```

---

## 输入一览

| 输入 | 提供内容 |
|------|---------|
| **(a) 潜在客户** | 公司、联系人、交易阶段、痛点、文字稿 |
| **(b) 受众** | 高管/技术/运营/混合 + 他们关注什么 |
| **(c) 目的** | 初次接触/探索跟进/深度交流/对齐/POC/成单 |
| **(d) 格式** | 落地页/PPT/单页纸/工作流演示 |

---

## 格式选择指南

| 如果您需要... | 选择... |
|--------------|---------|
| 令人印象深刻的多标签体验 | **交互式落地页** |
| 在会议中演示的内容 | **PPT 风格** |
| 快速留存的摘要 | **单页纸** |
| 系统如何连接的可视化 | **工作流演示** |

---

## 示例提示

**简单版：**
```
Create an asset for Acme Corp
```

**带背景版：**
```
Create an asset for Acme Corp. They're a manufacturing company
struggling with supply chain visibility. Met with their COO
last week. Need something for the exec team.
```

**工作流演示版：**
```
Mock up a workflow for Centric Brands showing how they'd use
our product to monitor contract compliance. Components: our AI,
their Snowflake warehouse, and scanned PDF contracts.
```

## 构建完成后

| 想要... | 说... |
|---------|-------|
| 更改颜色 | "Use our brand colors instead" |
| 添加章节 | "Add a section on security" |
| 缩短内容 | "Make it more concise" |

| 获取 PDF | "Give me a print-friendly version" |

---

## 输出

- 独立 HTML 文件
- 可离线使用
- 可托管于任意位置（Netlify、Vercel、GitHub Pages 等）
- 可通过托管服务商设置密码保护
