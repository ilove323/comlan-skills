# 创建销售资料

**适用于所有销售团队**

在数分钟内生成专业的、客户就绪的销售资料。无需设计技能。

---

## 功能说明

本技能通过询问您以下信息来创建定制销售资料：
1. **您的潜在客户** — 他们是谁，您们讨论了什么
2. **您的受众** — 谁在查看，他们关注什么
3. **您的目标** — 您想实现什么
4. **您的格式** — 以何种形式呈现

然后，技能负责调研、撰写、设计并构建精美的资料，可直接与客户分享。

---

## 支持的格式

| 格式 | 最适合 | 输出 |
|------|--------|------|
| **交互式落地页** | 高管会议、价值主张演示 | 带演示和计算器的多标签页面 |
| **PPT 风格** | 正式演示、大型受众 | 带导航的线性幻灯片 |
| **单页纸** | 留存资料、快速摘要 | 单屏滚动高管摘要 |
| **工作流/架构演示** | 技术深度交流、POC 提案 | 带动画流程的交互式图表 |

## 快速开始

### 选项一：简单提示
```
Create an asset for Acme Corp
```

### 选项二：带背景信息
```
Create an asset for Acme Corp. I met with their VP Engineering
last week - they're struggling with slow release cycles and
want to improve developer productivity. This is for a follow-up
with their technical team.
```

### 选项三：工作流演示
```
I want to mock up a workflow showing how a customer would use
our product to automate their invoice processing. The flow is:
invoices come in via email → our AI extracts data → validates
against their ERP → flags exceptions for human review.
```

## 创建内容

### 交互式落地页
- 标签页导航
- 公司指标和调研数据
- 使用场景演示
- ROI 计算器
- 专业深色主题，采用潜在客户的品牌色

### PPT 风格
- 含双方 Logo 的标题页
- 议程
- 内容幻灯片（每页一个核心信息）
- 摘要与下一步
- 包含演讲者备注

### 单页纸
- 主题陈述
- 3 个核心价值点
- 有力证明
- 清晰的 CTA

### 工作流演示
- 可视化组件节点
- 动画数据流
- 分步骤演示
- 播放/暂停/步进控制
- 每个阶段的样例数据

---

## 流程

```
1. 您提供背景（潜在客户、受众、目的）
          ↓
2. 技能调研潜在客户公司
          ↓
3. 技能提出 3-4 个确认问题
          ↓
4. 您确认方向
          ↓
5. 技能构建资料
          ↓
6. 您按需迭代修改
```

## 分享您的资料

输出是独立的 HTML 文件，分享方式：

- **静态托管**：上传至 Netlify、Vercel、GitHub Pages 或任意网络主机
- **密码保护**：大多数主机平台提供简单的密码保护
- **直接分享**：通过邮件发送 HTML 文件——可离线查看
- **嵌入**：可通过 iframe 嵌入其他页面或门户

## 获得最佳效果的建议

### 提供丰富背景信息
您分享的过往对话、痛点和干系人顾虑越多，资料就越贴合需求。

### 上传文字稿
如果有通话录音、会议记录或邮件往来，请上传。技能会提取关键引语和优先事项。

### 明确说明受众
"技术团队" 是好的。"正在评估我们安全模型的 IT 架构师" 更好。

### 自由迭代
第一稿不太理想？只需说明要改什么。颜色、章节、信息、流程——都可以调整。

## 常见问题

**问：适用于任何产品/公司吗？**
答：是的。技能会从您的邮件域名识别您所销售的产品，并相应调研。

**问：如何知道潜在客户的品牌色？**
答：从潜在客户的网站或品牌指南中提取。您可以事后调整。

**问：可以改用我们公司的品牌吗？**
答：可以——在初次构建后，只需要求切换到您的品牌色即可。

**问：如果调研结果有误怎么办？**
答：指出问题并提供更正信息。技能会用准确信息重新生成。

**问：可以导出为 PDF 吗？**
答：可以——请求打印优化版本，并使用浏览器的打印为 PDF 功能。
