# /policy-lookup

> 如遇不熟悉的占位符或需要查看已连接工具，请参见 [CONNECTORS.md](../../CONNECTORS.md)。

用通俗语言查询和解释公司政策，通过搜索已连接的知识库或使用提供的员工手册内容，回答员工关于政策、福利和流程的问题。

## 用法

```
/policy-lookup $ARGUMENTS
```

搜索与以下内容相关的政策：$ARGUMENTS

## 工作原理

```
┌─────────────────────────────────────────────────────────────────┐
│                    政策查询                                       │
├─────────────────────────────────────────────────────────────────┤
│  独立使用（始终可用）                                             │
│  ✓ 用通俗语言提问任何政策问题                                    │
│  ✓ 粘贴您的员工手册，AI助手将为您检索                           │
│  ✓ 获得清晰、无术语的解答                                        │
├─────────────────────────────────────────────────────────────────┤
│  增强模式（连接工具后）                                           │
│  + 知识库：自动搜索手册和政策文档                                │
│  + HRIS：获取员工个人信息（年假余额、福利）                      │
└─────────────────────────────────────────────────────────────────┘
```

## 常见政策主题

- **年假与休假**：带薪年假、病假、育儿假、丧假、长期休假
- **福利**：医疗保险、牙科、视力、养老金、HSA/FSA、员工健康
- **薪酬**：发薪周期、奖金时间节点、股权归属、报销流程
- **远程工作**：居家办公政策、异地工作、设备补贴、共享办公
- **差旅**：预订政策、每日补贴、报销申请、审批流程
- **行为规范**：行为准则、反骚扰政策、利益冲突
- **职业发展**：培训预算、会议政策、学费报销

## 回答方式

1. 在 ~~knowledge base 中搜索相关政策文档
2. 提供清晰、通俗易懂的答复
3. 引用具体的政策原文
4. 注明任何例外或特殊情况
5. 指引复杂问题的咨询对象

**重要注意事项：**
- 始终注明来源文档和章节
- 如未找到相关政策，明确告知而非猜测
- 对于法律或合规问题，建议直接咨询HR或法务

## 已连接的服务

若已连接 **~~knowledge base**：
- 自动搜索员工手册和政策文档
- 引用具体文档、章节和页码

若已连接 **~~HRIS**：
- 获取员工个人信息，如年假余额、福利选择和参保状态

## 使用建议

1. **用通俗语言提问** ——"我能在欧洲工作一个月吗？"比"国际远程办公政策"更好用。
2. **描述具体情况** ——"加州兼职员工的年假政策"比"年假政策"得到的答复更精准。
