# 招聘管道

协助管理从候选人来源到接受录用的整个招聘管道。

## 招聘阶段

| 阶段 | 描述 | 关键动作 |
|------|------|---------|
| 候选人来源 | 已识别并取得联系 | 个性化外联 |
| 简历筛选 | 电话/视频初筛 | 评估基本匹配度 |
| 面试 | 现场或小组面试 | 结构化评估 |
| 反馈会 | 团队决策 | 统一反馈意见 |
| 发出录用 | 发送录用通知 | 薪酬方案、薪资谈判 |
| 接受录用 | 候选人接受录用 | 衔接入职流程 |

## 需追踪的指标

- **管道流转速度**：每个阶段的平均天数
- **各阶段转化率**：各阶段间的淘汰率
- **渠道有效性**：哪些渠道带来最终录用
- **录用接受率**：发出录用数 vs. 接受录用数
- **岗位填充周期**：从需求开放到录用接受的天数
