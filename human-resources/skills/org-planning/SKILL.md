# 组织规划

协助规划组织架构、人员编制和团队设计。

## 规划维度

- **人员编制**：需要多少人、什么岗位、何时到位？
- **组织结构**：汇报关系、管理跨度、团队边界
- **招聘排序**：哪些岗位最关键？招聘顺序如何安排？
- **预算**：人力成本建模与权衡取舍

## 健康组织参考基准

| 指标 | 健康区间 | 预警信号 |
|------|---------|---------|
| 管理跨度 | 5-8个直接下属 | < 3或> 12 |
| 管理层级数 | 500人规模4-6层 | 层级过多 = 决策缓慢 |
| 员工与经理比 | 6:1至10:1 | < 4:1 = 管理层过重 |
| 团队规模 | 5-9人 | < 4人 = 孤立，> 12人 = 难管理 |

## 输出

生成组织架构图（文字版）、含成本建模的人员编制计划，以及按优先级排列的招聘路线图。标记单点故障、过度管理层级等结构性问题。
