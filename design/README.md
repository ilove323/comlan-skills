# 设计插件

这是一款主要为 [OpenClaw](https://openclaw.ai) 设计的效率插件——同时也适用于 Claude Code。帮助处理设计评审、系统管理、UX 写作、无障碍审查、研究合成和开发交接。适用于任何设计团队——可单独使用（提供您的描述或截图），连接 Figma 等工具后效果更佳。

## 安装

```bash
openclaw skills install design
```

## 命令

通过斜杠命令显式调用的工作流：

| 命令 | 描述 |
|---|---|
| `/critique` | 获取结构化设计反馈——可用性、视觉层次、无障碍性和一致性 |
| `/design-system` | 审计、记录或扩展您的设计系统——组件、令牌、模式 |
| `/handoff` | 生成开发交接规格——尺寸、令牌、状态、交互和边缘情况 |
| `/ux-copy` | 撰写或审查 UX 文案——微文案、错误信息、空状态、引导流程 |
| `/accessibility` | 运行无障碍审计——WCAG 合规、色彩对比、屏幕阅读器和键盘导航 |
| `/research-synthesis` | 合成用户研究——访谈、调查、可用性测试，转化为可操作洞察 |

所有命令均可**单独使用**（描述您的设计或粘贴截图），连接 MCP 连接器后效果**大幅提升**。

## 技能

AI助手在相关时自动使用的领域知识：

| 技能 | 描述 |
|---|---|
| `design-critique` | 评估设计的可用性、视觉层次、一致性和设计原则遵循情况 |
| `design-system-management` | 管理设计令牌、组件库和模式文档 |
| `ux-writing` | 撰写有效的微文案——清晰、简洁、一致且符合品牌调性 |
| `accessibility-review` | 审计设计和代码是否符合 WCAG 2.1 AA 规范 |
| `user-research` | 规划、开展和合成用户研究——访谈、调查、可用性测试 |
| `design-handoff` | 从设计稿创建全面的开发交接文档 |

## 示例工作流

### 获取设计反馈

```
/critique
```

分享 Figma 链接、截图或描述您的设计。获取关于可用性、视觉层次、一致性和无障碍性的结构化反馈。

### 审计设计系统

```
/design-system audit
```

AI助手将审查您的组件库的一致性、完整性和命名规范，并给出具体改进建议报告。

### 撰写 UX 文案

```
/ux-copy error messages for payment flow
```

获取符合上下文的文案，附带语气指导、备选方案和本地化说明。

### 开发交接

```
/handoff
```

分享 Figma 链接，获取完整规格：尺寸、设计令牌、组件状态、交互说明和边缘情况。

### 无障碍检查

```
/accessibility
```

分享设计或 URL。获取 WCAG 2.1 AA 合规报告，包含具体问题、严重程度和修复步骤。

### 合成研究

```
/research-synthesis
```

上传访谈记录、调查结果或可用性测试笔记。获取主题、洞察和优先建议。

## 单独使用 + 连接增强

每个命令和技能均无需集成即可使用：

| 可执行操作 | 单独使用 | 连接后增强 |
|-----------------|------------|-------------------|
| 设计评审 | 描述或截图 | Figma MCP（直接获取设计稿） |
| 设计系统 | 描述您的系统 | Figma MCP（审计组件库） |
| 交接规格 | 描述或截图 | Figma MCP（精确尺寸、令牌） |
| UX 文案 | 描述上下文 | 知识库（品牌声音指南） |
| 无障碍 | 描述或截图 | Figma MCP、真实使用数据分析 |
| 研究合成 | 粘贴记录 | 用户反馈工具（获取原始数据） |

## MCP 集成

> 如果您看到不熟悉的占位符或需要检查已连接的工具，请参阅 [CONNECTORS.md](CONNECTORS.md)。

连接您的工具以获得更丰富的体验：

| 类别 | 示例 | 功能 |
|---|---|---|
| **设计工具** | Figma | 获取设计稿、检查组件、访问设计令牌 |
| **用户反馈** | Intercom、Productboard | 原始反馈、功能请求、NPS 数据 |
| **项目追踪器** | Linear、Asana、Jira | 将设计链接到工单、追踪实施 |
| **知识库** | Notion | 品牌指南、设计原则、研究仓库 |
| **产品分析** | Amplitude、Mixpanel | 用于研究合成和设计决策的使用数据 |

完整集成列表请参阅 [CONNECTORS.md](CONNECTORS.md)。
